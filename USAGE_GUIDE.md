# FRAME 使い方

## 分析

1. 日本株または米国株を選択します。
2. 銘柄コードを入力します。
   - 日本株: `285A`、`5803`など
   - 米国株: `MU`、`NVDA`、`LRCX`など
3. 「分析する」を押します。

## 状態

- **WAIT**: 条件不足。監視のみ
- **READY**: 上位足が改善し、トリガーへ接近
- **TRIGGERED**: トレンド・出来高・相対強度が揃った状態
- **INVALID**: 日足と週足の下降が重なり、シナリオ再構築が必要

## 見る順番

1. 月足: 長期シナリオが維持されているか
2. 週足: 調整か下降転換か
3. 日足: 実際のエントリー条件が近いか
4. 相対強度: 指数より強いか
5. 出来高: 値動きに裏付けがあるか
6. 無効化条件: どこで考えが間違いになるか

## プラン保存

分析結果の「プラン保存」を押すと、プランタブへ保存されます。保存後も再分析できます。

保存するのは注文ではありません。FRAMEは自動売買を行いません。
