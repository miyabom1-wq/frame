# FRAME UI v0.12 使い方

## FRAMEの役割

VANTAGEが市場・テーマ・候補・信用需給を整理し、FRAMEが個別銘柄の局面、仕掛け位置、無効化条件を計算します。FRAMEは注文を送りません。

## 基本手順

1. 新規・押し目・長期保有から分析モードを選ぶ
2. 局面が上昇、押し目、修復、分配、崩壊のどれか確認
3. 打診・標準・追加の3段階を確認
4. 無効化条件を先に固定
5. 必要な銘柄をプラン保存

## VANTAGEから受け取る信用需給

日本株をVANTAGEから開くと、分析画面上部に次が表示されます。

- 需給追い風 / 改善 / 中立 / 悪化 / 警戒
- 信用倍率
- 信用買残
- 買残前週比
- 買残回転日数
- 週次基準日
- 日々公表、特別周知、信用規制

信用需給はVANTAGEの補助評価です。FRAMEの価格構造とエントリー水準は、取得した株価履歴から独立して計算します。

- 悪化・警戒：打診を小さくし、標準・追加は価格確認を優先
- 規制・特別周知：追加停止の注意情報
- 改善・追い風：テクニカル条件が成立した場合だけ補助材料にする

## 株価表示

最新取得価格、前日比、価格時刻を表示します。場中値はリアルタイム保証がないため、価格時刻を確認してください。「価格・判定を更新」で価格構造も再計算します。

## スマホへのインストール

右上のインストールボタン、またはChromeのメニューから「アプリをインストール」を選びます。旧FRAMEが残っている場合はホーム画面から削除し、Chromeを完全終了してから開き直します。
