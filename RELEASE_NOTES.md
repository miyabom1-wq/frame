# FRAME v0.3.0

- 単一の予定エントリーを廃止し、打診・標準・追加の3段階へ分離
- 打診は5日線・EMA21・5日戻り高値の最初の回復
- 標準はEMA65・25日線・10日戻り高値の回復
- 追加は20日高値突破、出来高比1.2以上、20日相対強度プラス
- 各段階に独立した撤回・再評価水準を追加
- 保存プランでも3水準を保持
- 既存の新規判定・保有判定・VANTAGEとの役割分担は維持
