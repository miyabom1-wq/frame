# FRAME v0.4.0

- 個別銘柄の価格局面を8分類
  - モメンタム初動
  - 上昇継続
  - 健全な押し目
  - ベース形成
  - 分配警戒
  - 修復中
  - 崩壊・下降
  - 移行・判定待ち
- 60日高値からの下落率、20日値幅、分配日、安値切り上げを局面判定へ追加
- 通常の押し目と、大幅下落後の修復局面を分離
- 分配警戒中は新規エントリーを停止
- 修復中は打診サイズを抑え、安値切り上げまたは相対強度改善を要求
- 保有判定を厳格化
  - 大幅ドローダウン
  - 20日相対強度の大幅悪化
  - 修復・分配局面
  を `CAUTION` に反映
- 保存プランへ局面判定を保存
- プラン一覧へ局面ラベルを表示

# Release Notes

## v0.3.0

- 単一の予定エントリーを廃止し、打診・標準・追加の3段階へ分離
- 打診は5日線・EMA21・5日戻り高値の最初の回復
- 標準はEMA65・25日線・10日戻り高値の回復
- 追加は20日高値突破、出来高比1.2以上、20日相対強度プラス
- 各段階に独立した撤回・再評価水準を追加
- 保存プランでも3水準を保持

## v0.2.0

- 新規エントリー判定と既存保有判定を分離
- 保有判定 `HOLD / CAUTION / REVIEW` を追加
- エントリー条件8項目のチェックリストと達成数を追加
- 相対強度を5日・20日・60日に拡張
- 新規・押し目・長期保有の分析モードを追加
- プランメモと分析モードをKVへ保存
