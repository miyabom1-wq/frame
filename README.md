# FRAME v0.2.0

個別銘柄のテクニカル構造を整理し、新規エントリーと既存保有を分けて判断するアプリです。

## 役割

- VANTAGE: 市場環境、資金フロー、候補抽出
- FRAME: 個別銘柄の時間軸、エントリー条件、保有条件、無効化条件

## v0.2.0の中心機能

- 月足・週足・日足の分離判定
- 新規判定 `WAIT / READY / TRIGGERED / INVALID`
- 保有判定 `HOLD / CAUTION / REVIEW`
- エントリー条件8項目の達成状況
- 5・20・60日相対強度
- 新規・押し目・長期保有モード
- プランとメモのKV保存

FRAMEは注文を送信しません。
